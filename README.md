# Investment Desk Pro

A responsive, self-contained professional trading information platform. It includes portfolio KPIs, performance comparison, multi-asset category allocation, a cross-asset technical indicator matrix, holdings, signal radar, market regime notes, and risk indicators.

## Open locally

Open `index.html` in a browser. No build step is required.

## Files

- `index.html` - dashboard markup
- `styles.css` - layout and visual styling
- `script.js` - dashboard data rendering and chart drawing
